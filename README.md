# Jiawen Nova Defense | 嘉文新星防御

A classic Missile Command style tower defense game built with React, Vite, and Tailwind CSS. Defend your cities from falling ICBMs!

## Features

- **ICBM Defense**: Intercept incoming enemy missiles before they hit your cities.
- **Tracking Missiles**: Earn special tracking missiles every 10 seconds to automatically target the most dangerous threats.
- **Dynamic Difficulty**: The game gets harder as your score increases.
- **City Skyline**: Defend a detailed city skyline background.
- **Bilingual Support**: Toggle between English and Chinese.

## Deployment

### Deploy to Vercel

1. Push this code to a GitHub repository.
2. Go to [Vercel](https://vercel.com) and import your repository.
3. Vercel will automatically detect the Vite project and deploy it.

### Local Development

```bash
npm install
npm run dev
```

## How to Play

1. **Click/Touch**: Fire an interceptor missile to the cursor position.
2. **Tracking Missile**: Click the "Tracking Missile" button to launch an auto-targeting missile.
3. **Goal**: Reach 1000 points to win. Don't let all your batteries be destroyed!
